How to Join

1

Install CurseForge

Download the CurseForge App if you don't have it already.

2

Download the Modpack

Click the download button above to get the Grudge Factions modpack zip (120 mods, Forge 47.4.0).

3

Import into CurseForge

Open CurseForge → Minecraft → Create Custom Profile → Import → Select the downloaded zip.

4

Launch & Connect

Hit Play in CurseForge. The server is pre-added to your server list. Click Join Server, then type /classstart to pick your class!
